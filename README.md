# RIYAA MAKEOVERS

Premium static website for a luxury beauty salon and makeup academy.

## Stack

- HTML5
- CSS3
- Vanilla JavaScript
- No Bootstrap, Tailwind, jQuery, or React

## Pages

- `index.html`
- `about.html`
- `services.html`
- `academy.html`
- `gallery.html`
- `testimonials.html`
- `contact.html`

## Features

- Sticky transparent navigation with mobile hamburger
- Dark mode toggle
- Scroll progress, animated cursor glow, loading screen, and back-to-top control
- Booking popup and success toast
- Animated counters, reveal effects, parallax, ripple buttons, and hover lift cards
- Services grid generated from reusable JavaScript data
- Pinterest-style gallery with filters and lightbox
- FAQ accordion, testimonial slider, contact form, and social action buttons

Open `index.html` in a browser to view the site.
